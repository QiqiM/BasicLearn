### 安装依赖

```shell
cd demo
npm install
```

### GRPC基本使用案例

```shell
cd basicGrpc
node server.js
node client.js
```

### Server Streaming GRPC使用案例

```shell
cd streamGrpc
node server.js
node client_grpc_server_stream.js
```

### Client Streaming GRPC使用案例

```shell
cd streamGrpc
node server.js
node client_grpc_client_stream.js
```
